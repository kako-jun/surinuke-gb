# surinuke-gb

ゲームボーイ用アスタリスク回避シューティングゲーム

## 概要

`surinuke-gb`は、ゲームボーイ向けのシンプルなアクションゲームです。プレイヤーは画面下部で左右のみに移動し、上から降ってくるアスタリスク（*）を避け続けます。

縦長の画面レイアウトで、カタカタという効果音とともにシンプルな操作性を追求したミニマルなゲームです。

## ゲームの遊び方

- **十字キー左右のみ**: プレイヤー（&マーク）を左右に移動
- **自機**: アンパサンド記号（&）※GBCではシアン色、GBではモノクロ
- **敵**: アスタリスク（*）※GBCでは赤色、GBではモノクロ
- **画面**: 縦長レイアウト、背景は真っ黒、左右に壁
- **目的**: 上から降ってくるアスタリスクを避け続ける
- **ゲームオーバー**: アスタリスクに当たるとゲームオーバー
- **スコア**: 通り抜けた（避けた）アスタリスクの数（画面下部に表示）
- **リトライ**: ゲームオーバー後、どのボタンでもゲームを再開可能

## ビルド方法

### ローカル環境でのビルド

#### 必要なもの

- [GBDK (GameBoy Developers Kit)](https://github.com/gbdk-2020/gbdk-2020)

#### コンパイル

```bash
make
```

これで`surinuke-gb.gb`ファイルが生成されます。

#### クリーンアップ

```bash
make clean
```

### Dockerを使用したビルド

Dockerがインストールされている環境では、以下のコマンドでビルドできます：

```bash
docker build -t surinuke-gb-builder .
docker run --rm -v $(pwd):/work surinuke-gb-builder
```

### GitHub Actionsによる自動ビルド

このリポジトリはGitHub Actionsによる自動ビルドに対応しています。`main`または`master`ブランチへのプッシュ時に自動的にビルドされ、生成されたROMファイルはArtifactとしてダウンロード可能です。

## 実行方法

生成された`surinuke-gb.gb`ファイルをゲームボーイエミュレータで実行してください。

推奨エミュレータ：
- [BGB](https://bgb.bircd.org/)
- [mGBA](https://mgba.io/)
- [SameBoy](https://sameboy.github.io/)

## 技術仕様

- **開発言語**: C (GBDK使用)
- **ターゲット**: Game Boy / Game Boy Color 両対応
  - **Game Boy Color**: カラー表示
  - **通常のGame Boy**: モノクロ表示
- **スプライト数**: 2個（プレイヤー1個 + アスタリスク1個）
- **解像度**: 160x144ピクセル（縦長レイアウト）
- **ゲーム領域**: 画面中央の約64ピクセル幅
- **カラー** (Game Boy Colorで表示時):
  - 背景: 黒
  - 自機: シアン（明るく目立つ）
  - 敵: 赤（危険な色で目立つ）
  - 壁: グレー（控えめ）
  - スコア: 黄色（見やすい）
  - ※通常のGame Boyではモノクロ（白黒）で表示されます
- **サウンド**: カタカタという効果音
