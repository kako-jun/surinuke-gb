# surinuke-gb

Game Boy / Game Boy Color 用のレーン制回避アクションゲーム。8列のレーンが並ぶ通路で、上から降ってくるアスタリスク（*）を自機（&）ですり抜け続ける。

## スクリーンショット

> ビルド後に BGB や SameBoy で実行すると、GBCモードではシアン・赤・グレー・黄のカラー表示。DMGモードではモノクロで動作。

## 遊び方

| 操作 | 機能 |
|------|------|
| STARTボタン | タイトル画面からゲーム開始 |
| 十字キー左右 | 自機をレーン間で移動（押下中は滑らか、離すとスナップ） |
| 任意のボタン | ゲームオーバー後にリトライ |

- 8列のレーンにそれぞれ1体ずつ、上からアスタリスク（*）が降ってくる
- 同じレーン・同じ行に入ると衝突でゲームオーバー
- 障害物が画面下端を通過するとスコア+1
- タイトル画面の待ち時間で乱数シードが決まるため、毎回異なるパターン
- スコアが上がるほど落下が速くなる（20/50/100点で段階的に加速）

## ビルド

[GBDK-2020](https://github.com/gbdk-2020/gbdk-2020) v4.3.0 以上が必要。

### ローカルビルド

```bash
export GBDK_HOME=/path/to/gbdk
make
```

### Docker ビルド

```bash
docker build -t surinuke-gb-builder .
docker run --rm -v $(pwd):/work surinuke-gb-builder
```

生成された `surinuke-gb.gb` をエミュレータまたは実機で実行。

## 技術仕様

- **ROM形式**: Game Boy ROM (.gb)
- **対応機種**: DMG（初代Game Boy）/ GBC（Game Boy Color）
- **解像度**: 160×144
- **レーン**: 8列（タイル列6〜13）、壁はタイル列5と14
- **スプライト**: 7個使用（自機×1 + 敵×6のリングバッファ）
- **壁タイル**: インデックス128（GBDKフォントとの衝突回避）
- **サウンド**: APUチャンネル1による移動音・衝突音
- **開発環境**: C言語 / GBDK-2020 v4.3.0
