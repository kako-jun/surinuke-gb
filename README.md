# surinuke-gb

ゲームボーイ用アスタリスク回避シューティングゲーム

## 概要

`surinuke-gb`は、ゲームボーイ向けのシンプルなアクションゲームです。プレイヤーは画面下部で左右に移動し、上から降ってくるアスタリスク（*）を避け続けます。

## ゲームの遊び方

- **十字キー左右**: プレイヤーを移動
- **目的**: 上から降ってくるアスタリスクを避け続ける
- **ゲームオーバー**: アスタリスクに当たるとゲームオーバー
- **スコア**: 避けたアスタリスクの数がスコアになります
- **STARTボタン**: ゲームオーバー後、ゲームをリスタート

## ビルド方法

### 必要なもの

- [GBDK (GameBoy Developers Kit)](https://github.com/gbdk-2020/gbdk-2020)

### コンパイル

```bash
make
```

これで`surinuke-gb.gb`ファイルが生成されます。

### クリーンアップ

```bash
make clean
```

## 実行方法

生成された`surinuke-gb.gb`ファイルをゲームボーイエミュレータで実行してください。

推奨エミュレータ：
- [BGB](https://bgb.bircd.org/)
- [mGBA](https://mgba.io/)
- [SameBoy](https://sameboy.github.io/)

## 技術仕様

- **開発言語**: C (GBDK使用)
- **ターゲット**: Game Boy / Game Boy Color
- **スプライト数**: 最大6個（プレイヤー1個 + アスタリスク5個）
- **解像度**: 160x144ピクセル
