# surinuke-gb

ゲームボーイ用アスタリスク回避シューティングゲーム

## 概要

`surinuke-gb`は、ゲームボーイ向けのシンプルなアクションゲームです。プレイヤーは画面下部で左右のみに移動し、上から降ってくるアスタリスク（*）を避け続けます。

縦長の画面レイアウトで、カタカタという効果音とともにシンプルな操作性を追求したミニマルなゲームです。

## ゲームの遊び方

- **十字キー左右のみ**: プレイヤー（&マーク）を左右に移動
- **自機**: アンパサンド記号（&）で黄色に表示
- **敵**: アスタリスク（*）で白く表示
- **画面**: 縦長レイアウト、背景は真っ黒
- **目的**: 上から降ってくるアスタリスクを避け続ける
- **ゲームオーバー**: アスタリスクに当たるとゲームオーバー
- **スコア**: 通り抜けた（避けた）アスタリスクの数
- **リトライ**: ゲームオーバー後、どのボタンでもゲームを再開可能

## ビルド方法

### 必要なもの

- [GBDK (GameBoy Developers Kit)](https://github.com/gbdk-2020/gbdk-2020)

### コンパイル

```bash
make
```

これで`surinuke-gb.gb`ファイルが生成されます。

### クリーンアップ

```bash
make clean
```

## 実行方法

生成された`surinuke-gb.gb`ファイルをゲームボーイエミュレータで実行してください。

推奨エミュレータ：
- [BGB](https://bgb.bircd.org/)
- [mGBA](https://mgba.io/)
- [SameBoy](https://sameboy.github.io/)

## 技術仕様

- **開発言語**: C (GBDK使用)
- **ターゲット**: Game Boy / Game Boy Color
- **スプライト数**: 2個（プレイヤー1個 + アスタリスク1個）
- **解像度**: 160x144ピクセル（縦長レイアウト）
- **ゲーム領域**: 画面中央の約64ピクセル幅
- **カラー**: 背景黒、自機黄色、敵白
- **サウンド**: カタカタという効果音
