# 主要機能

## 回避アクション

狭い壁に囲まれた通路の中で、上から降ってくる障害物を左右移動だけで避けます。障害物が画面下端を通り過ぎるたびにスコアが1点加算されます。操作は十字キーの左右のみ。覚えることは何もなく、触った瞬間に遊べます。

## 壁のある通路

画面の左右に壁があり、プレイヤーの移動範囲が制限されています。広い画面を自由に逃げ回れるのではなく、狭い空間で障害物をかわすスリルが生まれます。壁があることで「すり抜ける」感覚が強調されます。

## スコア表示

画面下部にリアルタイムでスコアが表示されます。避けた障害物の数がそのまま得点になるシンプルな仕組みです。「あと1つ避けられていれば」という悔しさが、次のプレイへの動機になります。

## 効果音

一定間隔でカタカタという効果音が鳴り、ゲームにリズム感を与えます。無音のゲームボーイ画面が、音が加わるだけで一気に「遊んでいる感」を生み出します。

## ゲームオーバーとリトライ

障害物に当たるとゲームオーバー画面が表示され、最終スコアが確認できます。任意のボタンを押すだけで即座にリトライできるため、「もう1回」の気持ちを途切れさせません。

## ゲームボーイカラー対応

ゲームボーイカラーで起動すると、自動的にカラー表示に切り替わります。自機はシアン、障害物は赤、壁はグレー、スコアは黄色で表示され、モノクロ版よりも視認性が上がります。通常のゲームボーイではモノクロで問題なく動作します。
