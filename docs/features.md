# 主要機能

## レーン制回避アクション

8列のレーン（タイル列6〜13）で構成された通路の中で、上から降ってくる障害物を左右移動で避けます。障害物が画面下端を通り過ぎるたびにスコアが1点加算されます。操作は十字キーの左右のみ。覚えることは何もなく、触った瞬間に遊べます。

## レーンスナップ移動

キーを押している間は滑らかにピクセル単位で移動し、キーを離すと最寄りのレーン中心に自動スナップします。アナログ的な操作感とデジタルな当たり判定の明快さを両立しています。

## 壁のある通路

画面の左右（タイル列5と14）に壁があり、プレイヤーの移動範囲が8レーンに制限されています。壁があることで「すり抜ける」感覚が強調されます。

## タイトル画面

起動すると「SURINUKE」と「Press START」が表示されるタイトル画面が出ます。STARTボタンを待っている間にハードウェアタイマー（DIV_REG）で乱数シードが撹拌されるため、毎回異なるパターンで障害物が出現します。

## スコア表示

画面下部中央にリアルタイムでスコアが表示されます。桁数に応じて中央寄せされるため、常に見やすい位置に表示されます。避けた障害物の数がそのまま得点になるシンプルな仕組みです。

## 段階的な難易度上昇

スコアに応じて落下速度が4段階で上昇します。

| スコア | 落下速度 |
|---|---|
| 0〜19 | 1 px/frame |
| 20〜49 | 2 px/frame |
| 50〜99 | 3 px/frame |
| 100〜 | 4 px/frame |

序盤は余裕があり、徐々にスリルが増していきます。

## 6体の敵スプライト（リングバッファ）

6体の敵スプライトがリングバッファ方式で画面上から順に降下します。各行に1体ずつ、ランダムなレーンに配置されます。画面下端を通過した敵は、最も上にいる敵の1行上にリサイクルされ、新しいレーンが割り当てられます。

## 効果音

移動時のクリック音、衝突時のヒット音がAPUチャンネル1で鳴り、ゲームにフィードバックを与えます。

## ゲームオーバーとリトライ

障害物と同じタイル行かつ同じレーンに入るとゲームオーバー。「GAMEOVER」と「Press Any Key」が表示され、任意のボタンで即座にリトライできます。

## ゲームボーイカラー対応

ゲームボーイカラーで起動すると、自動的にカラー表示に切り替わります。自機はシアン、障害物は赤、壁はグレー、スコアは黄色で表示され、モノクロ版よりも視認性が上がります。通常のゲームボーイではモノクロで問題なく動作します。
