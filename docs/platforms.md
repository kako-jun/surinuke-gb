# 対応環境

## 対応ハードウェア

| 機種 | サポート状況 |
|---|---|
| ゲームボーイ（DMG） | 対応（モノクロ表示） |
| ゲームボーイポケット | 対応（モノクロ表示） |
| ゲームボーイカラー（CGB） | 対応（カラー表示） |
| ゲームボーイアドバンス | 後方互換で動作 |

通常のゲームボーイではモノクロ4階調で表示されます。ゲームボーイカラーでは自動的にカラーパレットが適用され、自機・障害物・壁・スコアがそれぞれ異なる色で表示されます。

実機で遊ぶにはフラッシュカートリッジ（EverDrive GB 等）にROMを書き込む必要があります。

## 推奨エミュレータ

| エミュレータ | プラットフォーム | 備考 |
|---|---|---|
| BGB | Windows | 高精度なゲームボーイエミュレータ。デバッグ機能も充実 |
| mGBA | Windows / macOS / Linux | マルチプラットフォーム対応。GBC表示の再現性が高い |
| SameBoy | macOS / Windows / Linux | 高精度。macOSでの動作が安定 |

いずれのエミュレータでも、ROMファイル（`.gb`）を開くだけで起動できます。

## ビルド環境

ROMファイルは2通りの方法でビルドできます。

### ローカルビルド

ゲームボーイ向け開発キットがインストールされた環境で、プロジェクトルートから `make` を実行すると `surinuke-gb.gb` が生成されます。

### コンテナビルド

開発キットのインストールが不要な方法です。コンテナイメージをビルドし、その中でROMを生成します。ホストOSに依存せず、同じ結果が得られます。

## 出力ファイル

| ファイル | 説明 |
|---|---|
| `surinuke-gb.gb` | ゲームボーイ用ROMファイル。エミュレータまたは実機で動作 |

ファイルサイズは32KB以下です。ゲームボーイの標準的なROMサイズに収まっています。
