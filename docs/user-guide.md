# 使い方ガイド

## ゲームの始め方

ROMファイル（`surinuke-gb.gb`）をエミュレータで開くか、フラッシュカートリッジに書き込んで実機で起動します。起動するとタイトル画面が表示されます。

## タイトル画面

「SURINUKE」と「Press START」が表示されます。STARTボタンを押すとゲーム開始です。待っている間に乱数シードが生成されるため、毎回異なるパターンで障害物が出現します。

## 操作方法

| 操作 | 動作 |
|---|---|
| 十字キー左 | 自機を左に移動（滑らかにピクセル移動） |
| 十字キー右 | 自機を右に移動（滑らかにピクセル移動） |
| キーを離す | 最寄りのレーン中心にスナップ |
| STARTボタン（タイトル画面） | ゲーム開始 |
| 任意のボタン（ゲームオーバー時） | リトライ |

操作はこれだけです。上下移動やその他のボタン操作はありません。

## 画面の見方

### プレイ中

- **自機（&の形）**: 画面下部に表示。左右に動かして障害物を避けます
- **障害物（*の形）**: 8列のレーンにそれぞれ1体ずつ、上から降ってきます。自機と同じレーン・同じ行に入るとゲームオーバー
- **壁**: 画面の左右にある縦線（タイル列5と14）。この壁の内側8レーンだけが移動可能な範囲です
- **スコア**: 画面下部中央に数字で表示。避けた障害物の数です

### ゲームオーバー時

- 「GAMEOVER」の文字と最終スコアが画面に表示されます
- 「Press Any Key」の案内に従い、どのボタンでも押せばすぐに再スタートできます

## 遊び方のコツ

- 8つのレーンのどこかに障害物が降ってきます。自機と違うレーンにいればセーフです
- キーを押している間は滑らかに動き、離すとレーン中心にスナップします。細かい位置調整は不要です
- 壁ギリギリのレーンも使えます。端に寄ることで回避スペースを確保できます
- スコアが上がると落下速度が速くなります（20/50/100点で段階的に加速）

## ゲームボーイカラーでの違い

カラー対応機で起動すると自動的に色がつきます。設定や切り替えは不要です。

| 要素 | モノクロ | カラー |
|---|---|---|
| 自機 | 黒 | シアン |
| 障害物 | 黒 | 赤 |
| 壁 | 黒 | グレー |
| スコア | 黒 | 黄 |

カラー版では障害物が赤く目立つため、視認性が向上します。
