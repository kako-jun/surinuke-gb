# 使い方ガイド

## ゲームの始め方

ROMファイル（`surinuke-gb.gb`）をエミュレータで開くか、フラッシュカートリッジに書き込んで実機で起動します。起動すると即座にゲームが始まります。タイトル画面やメニューはありません。

## 操作方法

| 操作 | 動作 |
|---|---|
| 十字キー左 | 自機を左に移動 |
| 十字キー右 | 自機を右に移動 |
| 任意のボタン（ゲームオーバー時） | リトライ |

操作はこれだけです。上下移動やボタン操作はありません。

## 画面の見方

### プレイ中

- **自機（&の形）**: 画面下部に表示。左右に動かして障害物を避けます
- **障害物（*の形）**: 画面上部から降ってきます。当たるとゲームオーバー
- **壁**: 画面の左右にある縦線。この壁の内側だけが移動可能な範囲です
- **スコア**: 画面下部中央に数字で表示。避けた障害物の数です

### ゲームオーバー時

- 「GAME OVER」の文字と最終スコアが画面中央に表示されます
- 「Press Any Key」の案内に従い、どのボタンでも押せばすぐに再スタートできます

## 遊び方のコツ

- 障害物は一度に1つしか降ってきません。焦らず、落ち着いて位置を見極めましょう
- 壁ギリギリまで移動できます。端に寄ることで回避スペースを確保できます
- 障害物の落下位置はランダムです。パターンを読むより、反射で避ける遊びです

## ゲームボーイカラーでの違い

カラー対応機で起動すると自動的に色がつきます。設定や切り替えは不要です。

| 要素 | モノクロ | カラー |
|---|---|---|
| 自機 | 黒 | シアン |
| 障害物 | 黒 | 赤 |
| 壁 | 黒 | グレー |
| スコア | 黒 | 黄 |

カラー版では障害物が赤く目立つため、視認性が向上します。
