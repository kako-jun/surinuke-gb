# すり抜けGB — プロジェクト概要

## 何を作っているか

**すり抜けGB（surinuke-gb）** は、ゲームボーイ / ゲームボーイカラー向けの回避アクションゲームです。狭い通路の中で、上から降ってくる障害物をひたすら避け続けるだけのシンプルなゲーム。避けた数がそのままスコアになります。

## なぜ作ったのか

「ゲームボーイの実機で動くゲームを、自分の手で作ってみたい」——その純粋な好奇心が出発点です。

ゲームボーイは1989年に発売された携帯ゲーム機ですが、今でも世界中にファンがいて、自作ROMを実機やエミュレータで動かして楽しむ文化が続いています。制約の多いハードウェアだからこそ、「何ができるか」を考えること自体が面白い。

すり抜けGBは、その最初の一歩として「とにかく動くものを作る」ことを優先しました。ルールは極限まで単純に、操作は左右だけ。それでも、避けるたびにスコアが上がっていく快感と、「もう1回」と思わせる中毒性は、シンプルだからこそ生まれます。

## 解決したい課題

- **レトロゲーム開発の敷居を下げる**: 実際に動くゲームボーイROMを作り、その過程を残すことで「自分にもできそう」と思える事例を増やす
- **制約の中で遊びを見つける**: 160×144ピクセル、4色（GBCでも限られたパレット）という制約が、かえって創造性を引き出す
- **完成品を世に出す**: 「いつか作ろう」ではなく、小さくても完成したものを公開する

## 対象ユーザー

- レトロゲームが好きで、ゲームボーイの自作ROMに興味がある人
- エミュレータで手軽にレトロ風ゲームを遊びたい人
- 「自分もゲームボーイのゲームを作ってみたい」と思っている開発者
